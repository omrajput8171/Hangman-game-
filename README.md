A classic Hangman word guessing game built with Python.

## How to Play

1. Run the game:
   ```
   python hangman.py
   ```
2. Guess one letter at a time
3. You have 6 wrong guesses before the hangman is complete
4. Win by guessing all letters in the word!

## Features

- ASCII art hangman display
- 15 programming-themed words
- Tracks guessed letters
- Play again option
- Input validation
